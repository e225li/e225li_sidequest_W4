## Description

Based on example 3, creates and positions the player using preloads and loadJSON files, while adding dynamic tiles and obstacles using loops and arrays.

---

## Learning Goals

Learning Goals:

- Loads JSON levels (preload)
- Builds Level objects
- Creates/positions the Player
- Handles input + level switching
- Level.js (grid + drawing + tile meaning)
- Player.js (position + movement rules)
